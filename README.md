# PlayerVillager

PlayerVillager is a Fabric mod for Minecraft 26.1.2 that makes villagers behave like player-themed entities when they are renamed with a valid player name. Unnamed villagers keep their original villager model. Named villagers switch to a player model and try to use the corresponding player skin.

## Features

- Unnamed villagers render with the normal villager model.
- Villagers renamed with a player's name render with a player model.
- The mod attempts to resolve the named player's skin and falls back to the default player skin when the skin cannot be resolved.

## Requirements

- Minecraft 26.1.2
- Fabric Loader 0.19.3 or newer
- Fabric API 0.153.0
- Java 25
